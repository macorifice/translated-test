#### Installation & Usage
```
git clone https://github.com/macorifice/translated-test.git
```

## start mysql instance at localhost
```
docker-compose up
```

## start server API
```
npm run start
```

## run tests with Jest
```
npm run test
```

## run lint with ESLint
```
npm run lint
```
